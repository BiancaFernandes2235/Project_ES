1. Download the project folder.
2. Open Eclipse.
3. Go to **File > Open Projects from File System...**
4. Select the `Hospital` folder.

To start the console application:

1. In the Project Explorer, navigate to the **`src/main`** folder.
2. Locate and right-click on **`Main.java`**.
3. Select **"Run As"** > **"Java Application"**.
4. The application will start in the **Console** tab at the bottom.

## ðŸ§ª How to Run Tests

To verify the logic using the automated test suite:

1. In the Project Explorer, navigate to the **`src/test`** folder.
2. Right-click on the **`test`** package.
3. Select **"Run As"** > **"JUnit Test"**.
4. A "JUnit" tab will open showing a **Green Bar** (Success) or **Red Bar** (Failure).
